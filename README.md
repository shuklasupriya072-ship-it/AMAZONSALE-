# 🛒 Amazon Sales Analysis - SQL Project

## 📌 About Project
Analyzed 500+ Amazon India sales transactions to find key business insights using SQL and Power Query.

## 🛠️ Tools Used
- SQL (SQLite) — Data Analysis
- Power Query (Excel) — Data Cleaning
- OneCompiler — SQL Editor
